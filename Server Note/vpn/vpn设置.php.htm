常用工具/安装手续/PacketixVPN Client from home †

Packetix VPN Client Download :
http://vpn.cc.tsukuba.ac.jp/

↑

設定方法 †

↑

自宅から192.168.200.xへアクセスするための方法です。 †

Packetix VPNを設定します。

Host : newvpn.a-it.jp
Port : 8888 or 443
HUB名 : ChinaNewVPN
UserName? : china
Password : china123456

↑

IPアドレス設定 †

以下のIPアドレスを仮想LANカードに設定してください。
※IPアドレスの重複を防止するために、利用したら名前を記入してください。

192.168.200.230
192.168.200.231
192.168.200.232
192.168.200.233
192.168.200.234  (xzh used)
192.168.200.235
192.168.200.236  (wll used)
192.168.200.237  (sx used)
192.168.200.238
192.168.200.239  (ww used)

↑

接続後の注意点 †

自宅に居るときは、

route delete 192.168.200.0 mask 255.255.255.0 192.168.75.200

を実行してください。

会社に戻ったら

route add -p 192.168.200.0 mask 255.255.255.0 192.168.75.200

で設定を戻してください。

↑

注意点 †

社内に居るときはVPN接続をOffにしてください。
