legend

排列方向（orient）
图例列表的布局朝向
宽高（width height）
图例组件的宽高
位置（left top right bottom）
图例组件离容器上侧、下侧、左侧、右侧的距离
图标（icon）
图例项的 icon
图标尺寸（itemWidth itemHeight）
图例标记的图形宽度和高度
图例项间隔（itemGap）
图例每项之间的间隔。横向布局时为水平间隔，纵向布局时为纵向间隔。
图例选择的模式（selectedMode）
控制是否可以通过点击图例改变系列的显示状态。默认开启图例选择，可以设成 false 关闭。
除此之外也可以设成 'single' 或者 'multiple' 使用单选或者多选模式。

Legend不显示的问题

  • legend 中的 name 要和 series 中的 name 完全一致，否则 legend 就无法显示；
  • 用 endcode 设置数据时，要设置 seriesName, 否则 legend 就无法显示；
            		
//数据格式
"dataM9|2-55":[
    {
        "name": "@cword(1,8)驾校",
        "perK2|0-100": 50,
        "perK3|0-100": 50
    }
]
//option 配置
series:[
    {
        encode:{
            x:[0],
            y:[1],
            seriesName: [1] //需要设置 seriesName, 否则 legend 无法显示
        }
    },{
        encode:{
            x:[0],
            y:[2],
            seriesName: [2] //需要设置 seriesName, 否则 legend 无法显示
        }
    }
]

Legend icon 颜色渐变

  • 在series中 单独设置color的值, 但是此处color 好像不支持 function 形式；
            		
//此示例在 bar 柱图中实验过，可行                        
var colors = [["#a0a3fe", "#8551ee"],["#02d7bd", "#00a3d2"]];                        
$.each(ecOptions.oBar.series, function(i){
    this.color = {
        type: 'linear',
        x: 0,
        y: 0,
        x2: 0,
        y2: 1,
        colorStops: [{
            offset: 0, color: colors[i][0]
        }, {
            offset: 1, color: colors[i][1]
        }],
        global: false
    }
})
