搜索

远程搜索（服务器端搜索）
queryParams
queryParamsType = 'limit'时，queryParams的参数为：
  • 每页显示的记录数
  • 偏移量（如果当前为第3页，每页显示的数量为10，则offset=20）
  • 搜索的字符串
  • 排序的字段名
  • asc/desc
queryParamsType != 'limit'时，queryParams的参数为：
  • 每页显示的记录数
  • 当前第几页
  • 搜索的字符串
  • 排序的字段名
  • asc/desc

注意：如果后台直接接收queryParams的参数，那么后台的参数名称必须和queryParams的参数名称一致； queryParams的参数会作为URL的参数传递到后台，如： http://localhost:3363/MockData/Pupils?sortName=drivingProgress&sortOrder=asc

本地搜索（即本地客户端搜索，也是插件自带搜索）
使用插件自带的搜索框(全局搜索)
					
//这仅仅是针对全局的模糊查询，要想针对某一列模糊查询可以在customSearch函数中定义						
$table.bootstrapTable({
	search: true
})
使用自定义的搜索框(全局搜索)
					
$table.bootstrapTable({
	search: true,
	searchSelector: "#searchText" //"#searchText"为搜索框的ID
})
使用自定义的搜索框(某列的模糊搜索)
					
//只对 studentName 进行模糊搜索
$table.bootstrapTable({
	search: true,
	searchSelector: "#searchText", //"#searchText"为搜索框的ID
	customSearch: function(data, text){
		return data.filter(function (row) {
		  return row.studentName.indexOf(text) > -1
		})
	}
})
