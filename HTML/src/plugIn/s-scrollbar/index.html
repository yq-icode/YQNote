IE7+

使用说明

不需要固定width、height

大多数滚动条插件需要固定 width、height 以计算滚动条的尺寸，此插件不需要，当然也可以为其设置。有以下几种方法设置width、height：

  • 为滚动条容器设置 max-height/max-width
  • 为滚动条容器设置 height/width，单位可以为 百分比%和像素px

支持3种类型的滚动条

只需要将 class = "scrollbar-inner/scrollbar-outer/scrollbar-external" 加到滚动条容器上即可。

此三种形式的区别没弄懂，第三种external的实例没弄出来。

inner
scrollbar is displayed over content (Apple Mac OS)
outer
scrollbar shifts content to display itself (Microsoft Windows scrollbars)
external
scrollbar is located in any place of your page. So, you can choose between Apple Mac and Windows style scrollbar or even place it outside of the container and style it as you wish.

Scrollbar Options

autoScrollSize

[true|false] (default: true)

根据容器尺寸自动计算滚动条的尺寸

disableBodyScroll

[true|false] (default: false)

如果设置为 true，当鼠标在滚动条容器上时，body 部分无法滚动

onInit

[function] (default: null)

滚动条初次初始化时的回调函数

onScroll

[function] (default: null)

当滚动条滚动时的回调函数

onUpdate

[function] (default: null)

计算滚动条尺寸时的回调函数

onDestroy

[function] (default: null)

滚动条被销毁时的回调函数
