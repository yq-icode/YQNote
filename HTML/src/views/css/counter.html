css计算器 counter

摘自：https://www.w3cplus.com/css3/css-counters.html

浏览器兼容
  • IE8+

属性讲解

counter-reset

  • identifier 必须，用来定义计数器的名称，这个名称可以自定义,但不能是CSS的关键词，比如none、inherit之类
  • integer 可选，设置调用计算数器时起始值，默认值为0；如果设置的值为1，那么计数从2开始；如果设置的值是-5，那么计数从-4开始，依此类推。
  • identifier integer之间用空格分隔开来
  • counter-reset属性可以包含一个或者多个计数器标识符，每个后面都可以跟一个整数值,如section 0 heading 0;

counter-increment

  • identifier 必须，计数器名称，调用counter-reset定义的计数器名称。
  • integer 可选，是一个整数值，可以是正整数，也可以是负整数。用来设置递增的值，如果取值为负值时，表示递减。默认值为1。

counter(identifier,ltype)

  • counter()函数接受两个参数，而且两参数之间使用逗号(,)来分隔。
  • identifier:必须，第一个参数是counter-increment定义的属性值identifier
  • ltype:可选，第二个参数是用来设置计数器的风格，有点类似于list-style-type。默认情况之下取值为十制，但你也可以重置这个样式风格，比如upper-roman或者upper-alpha等。counter()函数第二个值与列表中的list-style-type值相等
