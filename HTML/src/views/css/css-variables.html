CSS3 var() 变量讲解

操作 CSS 变量

设置 CSS 变量

HTML 标签中设置CSS变量

直接正常 CSS 语句一样在 style 属性中设置即可。

		                
<div id="box" style="--color: #cd0000;">
    <img src="mm.jpg" style="border: 10px solid var(--color);">
</div>

JS设置CSS变量

使用 setProperty() 方法

		                
<div id="box">
    <img src="mm.jpg" style="border: 10px solid var(--color);">
</div>

如果要想让 var(--color) 生效，执行下面 JavaScript 代码：

		                
box.style.setProperty('--color', '#cd0000');

获取 CSS 变量

使用 getPropertyValue() 方法

		                
box.style.getPropertyValue('--color');

删除 CSS 变量

		                
box.style.removeProperty('--color');
