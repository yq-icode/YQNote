min/max-width/height

IE7+
值为 inherit 时
当 min/max-width/height 的值为 inherit 时， IE7 不支持
当与 overflow: auto/scroll 结合使用时
max-width/height 与 overflow: auto/scroll 结合使用时， IE8 中会出现一些问题，具体是哪些问题不明；
input button/submit button/reset button
为 input button/submit button/reset button 添加 min-width 时，IE7 不支持
