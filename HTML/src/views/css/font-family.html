字体

字体格式

.ttf 格式(TureTpe)

IE9+; Firefox3.5+; Chrome4+; Safari3+; Opera10+; iOS Mobile Safari4.2+

.ttf字体是Windows和Mac的最常见的字体，是一种RAW格式，因此他不为网站优化。

.ttc 格式(TureTpe)

TTC是几个TTF合成的字库

.otf 格式(OpenType)

Firefox3.5+; Chrome4.0+; Safari3.1+; Opera10.0+; iOS; Mobile Safari4.2+

.otf字体被认为是一种原始的字体格式，其内置在TureType的基础上，所以也提供了更多的功能.

.woff 格式(Web Open Font Format)

IE9+; Firefox3.5+; Chrome6+; Safari3.6+; Opera11.1+

WOFF

.woff字体是Web字体中最佳格式，他是一个开放的TrueType/OpenType的压缩版本，同时也支持元数据包的分离.

.eot 格式(Embedded Open Type)

IE4+

.eot字体是IE专用字体，可以从TrueType创建此格式字体.

.svg 格式(SVG)

Chrome4+; Safari3.1+; Opera10.0+; iOS Mobile Safari3.2+

.svg字体是基于SVG字体渲染的一种格式.

@font-face

兼容性写法

	            	
@font-face {
    font-family: 'YourWebFontName';
    src: url('YourWebFontName.eot'); /* IE9 Compat Modes */
    src: url('YourWebFontName.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
             url('YourWebFontName.woff') format('woff'), /* Modern Browsers */
             url('YourWebFontName.ttf')  format('truetype'), /* Safari, Android, iOS */
             url('YourWebFontName.svg#YourWebFontName') format('svg'); /* Legacy iOS */
}

unicode-range

中文、英文应用不同的字体(1)

						
/* 常用中文字符的编码范围为 4E00-9FCB */							
@font-face {
	font-family: 'mixFont';
	src: url('fonts/STHeiti-Light.ttf');
	unicode-range: U+4E00-9FCB; /* 此编码范围内的字符是中文，会应用字体 STHeiti-Light.ttf */
} 
@font-face {
	font-family: 'mixFont';
	src: url('fonts/HelveticaThin.ttf');
}

中文、英文应用不同的字体(2)

你希望数字英文是Helvetica字体，中文是苹方或微软雅黑，直接把英文字体放在前面就可以了！

因为，据我所知，这些英文字体是没有中文字符集映射的，也就是，英文字体实际上对中文是没有任何作用的。考虑到font-family的字体解析是从前往后依次的，所以，自然而然上面的代码数字英文是Helvetica字体，中文是苹方或微软雅黑，完全不需要使用unicode-range做吃力不讨好的事情。

						
.font {
    font-family: Helvetica, 'Pingfang SC', 'microsoft yahei';
}

对特殊字符进行处理

						
@font-face {
  font-family: BASE;
  src: local('PingFang SC'),
       local("Microsoft Yahei");	
}
@font-face {
  font-family: quote;
  src: local('SimSun');	
  unicode-range: U+201c, U+201d; /* 对字符'双引号'单独设置字体 SimSun */
}

解决 Chrome 浏览器默认显示最小字号为 12px

用css3的新属性 transform:scale() 来达到对文本进行缩放，但是注意，它会对元素进行整体缩放，而这不是我们所希望的，我们只是想单独缩放字体，所以用一个标签把需要缩放的文本包起来，然后单独对他添加 transform:scale() 避免对其他元素影响。

此外，transform:scale() 是对尺寸的元素进行缩放，而内联元素没有尺寸，所以对内联元素不起作用，因为需要为内联元素设置 display：inline-block 属性。

	            	
/*这样设置可以让字号显示的更小*/
.demo {
 	font-size: 12px
 	transform: scale(0.8);
    transform-origin: left;
}
