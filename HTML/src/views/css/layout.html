两列，一列固定，一列自适应

左列固定，右列自适应－浮动法

兼容性：IE6+

	            	
	<div class=”leftSide”>左列</div>
	<div class=”rightSide”>右列</div>                        
	                        
	            
	            	
	.leftSide{
	    float:left;
	    width:200px;
	}
	.rightSide{
	    margin-left:200px;
	}

＜说明＞
不需要再为right设置float:right属性，也不需要为其设置宽度，它会自动占满父元素的宽度。

左列固定，右列自适应－绝对定位法

兼容性：IE6+

	            	
	<div class=”leftSide”>左列</div>
	<div class=”rightSide”>右列</div>                         
	                        
	            
	            	
	.leftSide{
	position:absolute;left:0;top:0;width:200px;
	}
	.rightSide{
	    margin-left:200px;
	}

右列固定，左列自适应－浮动法

兼容性：IE6+

	            	
	<div class=”rightSide”>右列</div>                        
	<div class=”leftSide”>左列</div>
	                        
	            
	            	
	.leftSide{
		margin-right:200px;
	}
	.rightSide{
	    float:right;
	    width:180px;
	}

＜说明＞
“rightSide右列”一定要放在”leftSide左列”的前面，否则无法达到这种效果。而且这样写时不用清除浮动，如果清除浮动（clearfix）了，在IE6中反而有问题，其它浏览器IE7/IE8/GOOGLE即使清除浮动了也还都正常；

右列固定，左列自适应－绝对定位法

兼容性：IE6+

	            	
	<div class=”rightSide”>右列</div>                        
	<div class=”leftSide”>左列</div>                        
	                        
	            
	            	
	.leftSide {
		margin-right:200px;
	    background:#CCCCCC;
	}
	.rightSide {
		position:absolute;
	    right:0;
	    top:0;
	    width:200px;
	    background:green;
	}

三列

左右固定，中间自适应－绝对定位法

兼容性：IE6+

	            	
	<div class=”leftSide”>左列</div>
	<div class=”center”>中间列</div>
	<div class=”rightSide”>右列</div>                         
	                        
	            
	            	
	.leftSide {
		position:absolute;
		top:0px;
		left:0px;
		width:100px;
		height:300px;
		border:2px solid #cccccc;
	}
	.rightSide{
		position:absolute;
		top:0px;
		right:0px;
		width:100px;
		height:300px;
		border:2px solid #cccccc;
	}
	.center{
		margin-left:104px;
		margin-right:104px;
		height:300px;
		border:2px solid #cccccc;
	}

左右固定，中间自适应－浮动法

兼容性：IE6+

	            	
	<div class="wrap"> 
	   <div class=”right”>这是右侧的内容 固定宽度</div>
	   <div class=”left”>这是左侧的内容 固定宽度</div>
	   <div class=”main”>中间内容，自适应宽度</div>
	</div>                        
	                        
	            
	            	
	.right{ 
	    width:200px;
	    float:right; 
	    background-color:#960;
	}
	.left{ 
	    width:150px;
	    float:left; 
	    background:#6FF;
	}
	.main{ 
	    margin-left:150px;
	    margin-right:200px; 
	    background-color:#9F3;
	}

注意：中间列一定要放在左右两列的后面

多个块元素不换行

法一

兼容性：IE6+

	            	
	.wrap { white-space:nowrap; }
	.wrap div {display:inline-block; height:100px; overflow:hidden; width:100px; }  
	.wrap div { *display:inline; }
	.wrap div p { white-space:normal; }

此种方法比较简单，但是由于对源代码中的空格浏览器处理方法不一样，会造成内部div之间的间距不等，产生此间距的原因，是标签之间的空格——元素标签间的空格和制表符会被压缩为1个空格显示，而对于这些空格的处理，浏览器存在差异。删除内部div之间的空格，可以解决问题。

法二

兼容性：IE8+

	            	
	.wrap { display:table; }
	.wrap div { border:1px solid; display:table-cell;}
	.wrap div p{ height:100px; margin:0; padding:0; width:100px; }

此种方法对于支持display:table-cell的浏览器，效果是统一的，但是对于table-cell类型的元素，在css的应用上有很多限制，例如margin无效、宽度和高度受同行和同列的其他单元格影响等。但是此方法可以实现内部div的等高，即不设定高度，高度由内容决定，同一行的div等高。同时，此方法对于 IE7- 无效，因此针对 IE7- 还是要使用上面的方法。

兼容性：IE6+
