高度自适应

根据内容自动调节高度，从而实现自适应。

最小自适应高度

在IE6环境下，其实height就几乎就等于是min-height，指定容器高度后，当内容变多时有自动撑大容器的特质。而IE7 IE8 Firefox不会，它们只是显示固定height值，当内容变多时会溢出容器.所以可以这样设置CSS:
兼容性：IE6+

                    	
.selector{height:auto !important;height:600px; min-height:600px;}

最大自适应高度

                    	
.selector{
	height:auto !important;
    height:250px;
    max-height:250px;
    overflow:hidden;
    overflow-y:auto;
}      
/* 因为IE6不支持min/max-width, min/max-height,所以针对IE6：*/
* html .selector{
	height:expression(this.height > 250 ? '250px' : auto)
}
