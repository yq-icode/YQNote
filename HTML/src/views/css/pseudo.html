伪类 伪元素

伪元素

content属性

插入字符

	                
.ele:before {
    content: '--';
}

插入图片

	                
.ele {
	position: relative;
}
.ele:before {
    content: '';
    background: url('../img/triangle_up.png') no-repeat top left;
    position: absolute;
    top: -15px;
    z-index: 2;
    width: 32px;
    height: 16px;
}

插入属性

	                
<a href="http://www.w3school.com.cn">W3School</a>
		            
	            
	                
a:after{content:"(" attr(href) ")"}
/*运行结果：W3School(http://www.w3school.com.cn)*/
