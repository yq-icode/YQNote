如何实现页面等比例缩放？

zoom 属性是 IE 浏览器的专有属性，Firefox 等浏览器不支持, 通过 zoom 和 scale 结合来实现兼容 IE 的缩放功能，IE: zoom:2, 非IE浏览器：transform:scale(2)

	                
//scaleSize为缩放比例	                	
if($(ifrm).parents('.mBlock').hasClass('fullScreen')){ //全屏时
	if(navigator.appName == "Microsoft Internet Explorer"){ //IE浏览器
		$(ifrm).parents(':root .mBlock').css({
			'zoom': scaleSize
		});
	}
	else{
		$(ifrm).parents('.mBlock').css({ //非IE浏览器
			'-webkit-transform': 'scale(' + scaleSize + ')',
			'-moz-transform': 'scale(' + scaleSize + ')',
			'transform': 'scale(' + scaleSize + ')'
		});
	}
}
else{ //取消全屏时
	$(ifrm).parents('.mBlock').css({
		'zoom':'',
		'-webkit-transform':'',
		'-moz-transform':'',
		'transform':''
	});
}
