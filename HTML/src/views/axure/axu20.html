select分组下拉框

遇到的问题

  • 根据元件内容判断，当元件内容为空时，隐藏元件，隐藏的元件还是占据空间
  • 根据元件内容判断，当元件内容为空时，设置元件的高度为0，高度为0的元件还是占据空间；

解决方法

并不是因为‘隐藏元件/高度为0的元件’占据了空间，而是该‘隐藏元件/高度为0的元件’的相邻元件的位置没有变动，需要移动其相邻的位置才能达到效果。
因为中继器中的元件，其X、Y坐标是以中继器为基准的，隐藏元件后，隐藏元件的X、Y坐标是不变的，只有移动元件，其X、Y坐标才会改变。

隐藏下拉框中的滚动条

原理分析

  • 设置动态面板 P1 的宽度：P1.width = 200
  • 设置可滚动的动态面板 P2 的宽度：P2.width = P1.width + 20（20为滚动条的宽度）
  • 按以上设置，P2 滚动条部分就被 P1 遮挡住了，这样就实现了隐藏下拉框滚动条的效果。
