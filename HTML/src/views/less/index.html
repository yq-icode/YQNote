box-shadow

以下以box-shadow为例。

此写法只适合一组参数 
                    
    .box-shadow(@x:0; @y:0; @blur-radius:0; @spread-radius:0; @color:#ccc; @rest…){
        -webkit-box-shadow:@arguments;
        -moz-box-shadow:@arguments;
        box-shadow:@arguments;
    }
// 参数间用‘分号’间隔

注意：因为外阴影是默认投影方式，而且没有默认值；只有是内阴影时才会加上inset，所以关于投影方式这个变量不能随便起个变量名，只能用@rest...，而且必须得放在最后。内阴影参数值也必须得相应的放在最后，这样参数值和变量才是一一对应的。

                    
    .btn4{.box-shadow(0,3px);}
    .btn4{.box-shadow(0,3px,0,0,#ccc);}
    .btn4{.box-shadow(0,3px,0,0,#ccc,inset);}
// 参数值用‘逗号’间隔；参数变量设置了默认值的可以省略不写；
此写法适用于一组及以上参数 
                    
    .box-shadow-multiple(@shadow){
        @shadow-double:@shadow;
        -webkit-box-shadow:@shadow;
        -moz-box-shadow:@shadow;
        box-shadow:@shadow;
    }            
                    
                
                    
    .btn4{
        ……
        @shadow:0 7px 0 0 #ccc, inset 0 8px 3px 0 rgba(0,0,0,.2);
        .box-shadow-multiple(@shadow);
    }

同系列颜色的定义

对于同系列颜色，如蓝色，不论是深蓝还是浅蓝，无论是字体颜色还是背景、边框等颜色，只需要定义一个颜色变量 @color:blue ，则其它的颜色可以通过 Color Operation Functions 来实现，如： darken(@color,20%),lighten(@color,50%)，这样的便利即是当需要将该蓝色系改成红色系时，只需要改 @color 的值就可以了，即：@color:red ，一步即完成转换。
