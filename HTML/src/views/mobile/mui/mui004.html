滚动

自定义滚动区域
自定义滚动区域垂直滚动时，需要为滚动区域设置高度；
自定义滚动区域水平滚动时，需要为滚动区域设置宽度；
自定义滚动区域 = mui-content区域 时
如果你的自定义区域就是mui-content区域，那么使用的时候，把mui-content和mui-scroll-wrapper放在一起就可以了。这样就不需要设置高度了。
                
//自定义滚动区域 != mui-content区域 时                	
<div id="scroll" class="mui-scroll-wrapper" style="min-height: 480px;">
    <div class="mui-scroll">
        ...
    </div>
</div>

//自定义滚动区域 = mui-content区域 时                	
<div id="scroll" class="mui-content mui-scroll-wrapper">
    <div class="mui-scroll">
        ...
    </div>
</div>

含有侧滑菜单的页面的滚动

当页面中包含侧滑菜单时，主页面的滚动会和侧滑菜单中的滚动有冲突，导致主页面无法实现滚动效果，此时为主页面的滚动添加以下代码即可解决：

                
mui('#manPg').scroll({
	deceleration: 0.0005
});

获取 scroll 控件当前位置的两种方式

                
//一、                	
var scrolly = mui('.mui-scroll-wrapper').scroll().y;

//二、添加事件监听scroll滚动时会触发该事件，监听事件需要放在mui.plusReady(function()内
document.querySelector('.mui-scroll-wrapper').addEventListener('scroll', function(e) {
	if(e.detail.y <= e.detail.maxScrollY){ //滚动到底部
	}
	if(e.detail.y >= 0){ //滚动到顶部
	}
});

//scroll控件中内容更新后，深度并不会随之更新，触发更新可以是手动滑动一下或者使用函数更新。（静态内容不存在该问题）
mui('.mui-scroll-wrapper').scroll().reLayout();
