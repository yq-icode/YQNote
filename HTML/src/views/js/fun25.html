JS 模块化介绍

JS 模块化规范有以下几种，在 ES6 以前，最主要的 JS 模块规范有两种：Node CommonJS 和 AMD，前者用于服务器，后者用于浏览器。

几种模块化的区别
Node CommonJS（服务端）
AMD(Asynchronous Module Definition 异步模块定义，客户端)

主要有两个Javascript库实现了AMD规范：require.js 和 curl.js。
