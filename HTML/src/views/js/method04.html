JS提交

但有时候我们会出于安全角度考虑，给用户输入的密码进行加密，方法一就没办法解决这个问题，这时候我们同常会选择另一种方法，使用javaScript来进行表单提交，代码入下：

	                
	<form action="" method="" id="test_form">
	    <input type="text" name="username" value="" />
	    <input type="password" name="pwd" value="" />
	    <button type="button" onclick='doSubmitForm()'>提交<button/>
	</form>
			        
	            
	                
	var form = document.getElementById('test_form');
	form.submit();

这种方法有个缺点就是，打乱正常的表单提交程序，通常用户输入完成后点击回车键就可以提交，但是这个方法实现不了，所以，使用下面的方法便可以解决这个问题,，通过form自身的onsubmit方法，来触发提交，然后进行input的修改：

onsubmit提交

	                
	<form id='test_form' action='' method='' onsubmit='return checkForm()'>
	    <input type='text' name='username' value=''/>
	    <input type='password' name='pwd' value =''/>
	    <button type='submit'>提交<button/>
	<form/>
			        
	            
	                
	function checkForm(){
	    var form = document.getElementById('test_form');
	    return ture;
	}

注意，checkForm()方法中，return true时，表单才会正常提交，为false时，浏览器将不会提交，通常是用户的密码输入错误时，终止提交。

修改提交的数据

表单只会提交带有 name 属性的表单元素

	                
	<form id='test_form' action='' method='' omsubmit='return checkForm()'>
	    <input type='text' name='username' value=''/>
	    <input type='password' id='input_pwd' value =''/>
	    <input type='hidden' name='pwd' id='md5_pwd' value=''/>
	    <button type='submit'>提交<button/>
	<form/>
			        
	            
	                
	function checkForm(){
	    var input_pwd= document.getElementById('input_pwd');
	    var md5_pwd= document.getElementById('md5_pwd');
	     md5_pwd.value= toMD5(input_pwd.value); //MD5密码加密方法       
	    //进行下一步
	    return ture;
	}

<input type='password'/>是用户输入密码的input框，并没有设置 name 属性，而是给 <input type='hidden' /> 设置了 name='pwd'，这样表单提交只会提交带有name属性的输入框，从而修改了表单提交的数据，只提交了我们想要的数据。

如何不提交隐藏DIV中的表单？

去除表单元素的 name 属性
此方法有效果（去除 name 属性之后，不需要重新渲染表单）
为表单元素添加 disabled 属性
通过为 select 添加 disabled 属性做测试，发现此方法在Layui框架中不起作用，无论是将 disabled 加在 select 原始DOM对象上，还是加在美化后的 input 标签上，都不起作用。

重置表单

	                
//法一: JS
document.getElementById("formId").reset();
	
//法二: JQUERY
$('#formId')[0].reset();
	
//法三: 使用input按钮
<input type="reset" name="reset" />

清空默认值

虽然reset方法可以清空表单，但是对有默认值的表单元素，只会让其恢复成默认值，而不会清空。

	                
<input name="percent" value="50"/> //重置后值变为：50

//如要清空，需要进行以下操作
$(':input','#myform').not(':button, :submit, :reset, :hidden').val('').removeAttr('checked').removeAttr('selected');
