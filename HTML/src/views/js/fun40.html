多个 checkbox 验证

验证要求：一组多个 checkbox 中必选一个

	                
<div class="custom-control custom-checkbox custom-control-inline">
	<input type="checkbox" class="custom-control-input" id="weekThis" name="weekThis" />
	<label class="custom-control-label" for="weekThis">This week
</div>
<div class="custom-control custom-checkbox custom-control-inline">
	<input type="checkbox" class="custom-control-input" id="weekNext" name="weekNext" />
	<label class="custom-control-label" for="weekNext">Next week
</div>
		            
	            
	                
if($("[name^='week'][checked]").length == 0){
	//验证信息
}
  • 获取 input:checked length, length > 1 即符合要求；
  • 为获取 name 方便，要有相同的命名规则，如上面 checkbox 的 name 都是以 'week' 开头；
