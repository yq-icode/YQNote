通过 jquery 的 getJSON()方法读取JSON数据

$.getJSON(url,data,success(data,status,xhr))
url
必需。规定将请求发送的哪个 URL。
data
可选。规定连同请求发送到服务器的数据。
success(data,status,xhr)
可选。规定当请求成功时运行的函数。
  1. response - 包含来自请求的结果数据
  2. status - 包含请求的状态
  3. xhr - 包含 XMLHttpRequest 对象

})
