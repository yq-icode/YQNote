Math方法

Math.floor(x)

	                   
Math.floor(x)
//example
Math.floor(0.60) --> 0
Math.floor(0.40) --> 0
Math.floor(-5.1) --> -6
Math.floor(-5.9) --> -6

floor()方法执行的是向下取整计算，它的返回值是：小于等于 x，且与 x 最接近的整数。

Math.ceil(x)
//example
Math.ceil(0.60) --> 1
Math.ceil(0.40) --> 1
Math.ceil(-5.1) --> -5
Math.ceil(-5.9) --> -5

ceil()方法执行的是向上取整计算，它的返回值是：大于等于 x，并且与它最接近的整数。

Math.round(x)

	                   
Math.round(x)
//example
Math.round(0.60) --> 1
Math.round(0.40) --> 0
Math.round(-5.1) --> -5
Math.round(-5.9) --> -6

round()方法可把一个数字舍入为最接近的整数。对于 0.5，该方法将进行上舍入。例如，3.5 将舍入为 4，而 -3.5 将舍入为 -3。

Math.abs(x)

abs()方法可返回数的绝对值。

Math.max(x...)

	                   
Math.max(x...)
//example
Math.max(7.25,7.30) --> 7.3
Math.max(5,8,6,-5,-6) --> 8
Math.max() --> Infinity
Math.max("dreamdu",8) --> NaN
Math.max(NaN,0) --> NaN
x : 0 或多个值。在 ECMASCript v3 之前，该方法只有两个参数。

max()方法可返回参数中最大的值。如果没有参数，则返回 Infinity。如果有某个参数为 NaN，或是不能转换成数字的非数字值，则返回 NaN。

parseInt(string, radix)

parseInt() 函数可解析一个字符串，并返回一个整数。

radix可选，表示要解析的数字的基数，该值介于 2 ~ 36 之间。
  • radix:如果省略该参数或其值为 0，则数字将以 10 为基础来解析。
  • radix:如果该参数小于 2 或者大于 36，则 parseInt() 将返回 NaN。
当参数 radix 的值为 0，或没有设置该参数时，parseInt() 会根据 string 来判断数字的基数。
  • 如果 string 以 "0x" 开头，parseInt() 会把 string 的其余部分解析为十六进制的整数。
  • 如果 string 以 1 ~ 9 的数字开头，parseInt() 将把它解析为十进制的整数。
  • 如果 string 以 0 开头，那么 ECMAScript v3 允许 parseInt() 的一个实现把其后的字符解析为八进制或十六进制的数字。
注意
  • 如果字符串的第一个字符不能被转换为数字，那么 parseInt() 会返回 NaN。
  • 只有字符串中的第一个数字会被返回。
  • 开头和结尾的空格是允许的。
	                   
	parseInt(string, radix)
	//example
	parseInt("10");         //十进制 10
	parseInt("19",10);      //十进制 10+9=19
	parseInt("11",2);       //二进制 2+1=3
	parseInt("17",8);       //八进制 8+7=15
	parseInt("1f",16);      //十六进制 16+15=31
	parseInt("010");        //未定：返回 10 或 8
	parseInt("He was 40");  //NaN

toFixed()

toFixed() 方法可把 Number 四舍五入为指定小数位数的数字。

	                   
	NumberObject.toFixed(num)
	//example
	var num = new Number(13.37); 
	num.toFixed(1) //13.4
num
必需。规定小数的位数，是 0 ~ 20 之间的值，包括 0 和 20，有些实现可以支持更大的数值范围。如果省略了该参数，将用 0 代替。
