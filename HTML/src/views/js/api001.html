宽度

关于jquery中对于宽度计算的几种方法

.width()

此方法有以下几种形式

.width()
描述

返回一个没有单位的数值（例如，400）

总是返回内容宽度,不管CSS box-sizing属性值
所以当计算元素占据空间的宽度时，如果元素设置有 box-sizing: border-box，需要加上border和padding值，才是元素占据空间的宽度。为了避免这种问题，请使用 $(ele).css('width')

其它

这个方法同样能计算出window和document的宽度

				                   
								$(window).width();
								$(document).width();
.width(value)
描述
给每个匹配的元素设置CSS宽度。
value
类型: String, Number
value 为 Number 时，jQuery会自动加上单位px；
value 为 String 时，任何有效的CSS尺寸都可以为宽度赋值（就像100px, 50%, 或者 auto）
注意
此方法设置的容器宽度是根据CSS box-sizing属性来定的.
如果元素设置了 border-box , 则此宽度包含 padding border;
如果元素未设置 border-box , 则此宽度不包含 padding border;
.width(function(index, width))

.innerWidth()

描述
为匹配的元素集合中获取第一个元素的当前计算宽度值,包括padding，但是不包括border。
这个方法不接受任何参数。
这个方法返回元素的宽度，包括左边和右边的padding，单位是像素。
其它
这个方法不适用于window 和 document对象，可以使用.width()代替。

.outerWidth([includeMargin ])

描述
获取元素集合中第一个元素的当前计算宽度值,包括padding，border和选择性的margin。（注：返回一个整数（不包含“px”）
返回元素的宽度，包括左右 padding值，border值和可选择性的margin。单位为像素。
includeMargin

一个布尔值，表明是否在计算时包含元素的margin值。
如果 includeMargin省略或者false，padding 和 border会被包含在计算中；如果true，margin也会被包含在计算中 。

其它
这个方法不适用于window 和 document对象，可以使用.width()代替。

示例

		                   
						$(ele).outerWidth(); //不包含 margin
						$(ele).outerWidth(false); //不包含 margin
						$(ele).outerWidth(true); //包含 margin

$(ele).css('width')

返回的是CSS中设置的宽度，带有完整单位的字符串（例如，400px）
