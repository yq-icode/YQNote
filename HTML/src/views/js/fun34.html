CSS 动画事件
animationstart
某个 CSS 动画开始时触发。
animationend
某个 CSS 动画完成时触发。
animationiteration
某个 CSS 动画完成后重新开始时触发。
CSS 过渡事件
transitionstart
A CSS transition has actually started (fired after any delay).
transitionend
A CSS transition has completed.
transitioncancel
A CSS transition has been cancelled.
transitionrun
A CSS transition has begun running (fired before any delay starts).
