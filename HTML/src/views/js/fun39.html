input[type=number]

设置 maxlength

<input type="text" maxlength="5" /> 效果ok，当 <input type="number" maxlength="5" /> 时 maxlength失效，长度可以无限输入。

解决方法 
	                
<input type="number" oninput="if(value.length > 5) value = value.slice(0,5)" />
  • maxLength 的默认值为 -1，即不设置 maxLength 属性值，maxLength 的值为 -1

设置 min & max

max，min的大小限制以及 step 属性是给右侧的上下按钮使用的，手动输入数值的话，它就是个input框，因此无法限定手动输入值的大小

设置 maxlength 及 min max

	                
<input type="number" oninput="if(value-max>0) value=max; if(value.length>maxLength) value=value.slice(0,maxLength);" max="9999.99" maxlength="7" >
  • value 与 max 进行比较时，如果判断写成 value > max 不起作用，value - max > 0 则可以，原因不明

input[type=text]

只能输入数字

					
<input type="text" oninput="value=value.replace(/[^\d]/g,'')" />
