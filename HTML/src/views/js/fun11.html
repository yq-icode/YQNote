$.data() $.removeData()讲解

$.data( element, key ) $.data( element, key, value )

element
Element类型 要存储数据的DOM对象
key
可选。String类型 指定的键名字符串。
value
可选。 Object类型 需要存储的任意类型的数据。
注意

通过 data() 函数存取的数据都是临时数据，一旦页面刷新，之前存放的数据都将被移除。

undefined 是一个没有被识别数据值。调用 jQuery.data( el, "name", undefined ) 将返回对应的 "name" 数据，等价于 jQuery.data(el, "name" ) 。

该方法目前并不提供在XML文档上跨平台设置，Internet Explorer不允许在XML文档中通过自定义属性附加数据。

$(selector).removeData(name)

name
可选。规定要删除的数据的名称。
如果没有规定名称，该方法将从被选元素中删除所有已存储的数据。

$.data()存储与匹配元素相关的任意数据

直接向元素附加数据

可以在一个元素上设置不同的值，并获取这些值

	                
//附加数据	                	
$("div").data("greeting1", "Hello World1");
$("div").data("greeting2", "Hello World2");

//返回数据
$("div").data("greeting1")//返回Hello World1
$("div").data("greeting2")//返回Hello World2

//删除数据
$("div").removeData("greeting1");
$("div").removeData();

使用对象向元素附加数据

	                   
testObj = new Object();
testObj.greetingMorn = "Good Morning!";
testObj.greetingEve = "Good Evening!";

$("div").data(testObj);//把数据添加到div元素
$("div").data("greetingEve")//获取已添加到div元素的数据： 返回Good Evening!

$(function () { 
	var div = $( "div" )[ 0 ];
	jQuery.data( div, "test", {
		first: 16,
		last: "pizza!"
	});
	$( "span:first" ).text( jQuery.data( div, "test" ).first );
	$( "span:last" ).text( jQuery.data( div, "test" ).last );
})
//运行结果：存储的值为 16 和 pizza!
