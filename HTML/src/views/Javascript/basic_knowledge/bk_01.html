history.go()与history.back()的区别

刷新:<input type="button" value="刷新" onclick="window.location.reload()" />
前进:<input type="button" value="前进" onclick="window.history.go(1)" />
后退:<input type="button" value="后退" onclick="window.history.go(-1)" />
前进:<input type="button" value="前进" onclick="window.history.forward()" />
后退:<input type="button" value="后退" onclick="window.history.back()" />
后退+刷新:
<input type="button" value="后退+刷新" onclick="window.history.go(-1);window.location.reload()" />
history.go(i)去指定的某页
history.go(0)刷新当前页

获取浏览器信息

浏览器的类型window.navigator.appName
IE版本window.navigator.appVersion
客户端分辨率window.screen.height; window.screen.width;
appCodeName：获取浏览器代码名称
userAgent：包含appName以及appVersion的所有信息
platform：获取运行浏览器的平台信息
language：获取浏览器支持的语言版本信息 
function checkIE(){
   var   X,V,N;   
   V=navigator.appVersion;   
   N=navigator.appName;   
   if(N=="Microsoft Internet Explorer")     
        X=parseFloat(V.substring(V.indexOf("MSIE")+5,V.lastIndexOf("Wind    ows")));
   else   
        X=parseFloat(V);   
   if(X<6.0){   
        alert("你的浏览器是：" + N + "\n" + "版本号：" + X + ".0");
  }
}
