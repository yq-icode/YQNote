缓冲返回页面顶部

缓冲效果的JS返回页首（网页顶部）特效，不是用的锚点技术，是用JS判断出来的，越出高度后会出现返回顶部的按钮，点击后页面由快到慢返回到网页顶端.
IE7,IE8,FF运行正常，IE6中缓冲效果也正常，只是返回顶部的按钮不会隐藏，会一直在页面底部。
注意：JS代码要写在<div id="gotopbtn">goTop</div>之后，否则无法取到其ID。 
<div id="gotopbtn">goTop</div>

<script type="text/javascript">
backTop=function (btnId){
	var btn=document.getElementById(btnId);
	var d=document.documentElement;
	window.onscroll=set;
	btn.onclick=function (){
		btn.style.display="none";
		window.onscroll=null;
		this.timer=setInterval(function(){
			d.scrollTop-=Math.ceil(d.scrollTop*0.1);
			if(d.scrollTop==0) clearInterval(btn.timer,window.onscroll=set);
		},10);
	};
	function set(){btn.style.display=d.scrollTop?'block':"none"}
};
backTop('gotopbtn');
</script>

<style type="text/css">
#gotopbtn {
	width:100px;
	height:30px;
	background:#fff;
	position:fixed;
	bottom:100px;
	right:20px;
	display:none;
	cursor:pointer;
	font-size:14px;
	line-height:30px;
	border:1px solid #aaa; 
}
