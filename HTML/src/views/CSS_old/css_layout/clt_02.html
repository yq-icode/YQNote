<html>

2.不同高度的两列间的分隔线

<style>
div{border:1px solid red;width:100px;height:80px;float:left;}
</style>
<div style="height:50px">454545</div>
<div style="margin-left:-1px;">454545</div>

原理：两个DIV都设置边框，然后让两个边框重叠；

3.侧栏高度问题

当我们在布局时，往往碰到左/右侧栏内容过少时，高度不能占满整个屏幕，而我们又需要这种效果，此时我们可以制作一张背景图片，将其作为body的背景图，并将侧栏的位置摆放在背景图中对应的位置，因为body的背景图是占满整个屏幕高度的，所以这种效果就实现了。

以obayashi为例，它的右侧栏的两条线就是通过在body中设置背景图片实现的。
