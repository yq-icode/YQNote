图片的圆角及垂直居中

说明

实现图片居中最好的容器就是表格，如果用其它容器实现的方法就比较复杂，再加上圆角效果，二者结合实现方法就更复杂了，因此我用了以下方法CSS+JS来实现图片的垂直居中以及圆角效果。

HTML

<div class="cornerBox">
  <table>
    <tr>
	  <td class="cornerBox_td">
		<img src="/images/css/3.jpg" alt="mainImg" />
	  </td>
	</tr>
  </table>
</div>

CSS

.cornerBox{
	position:relative;
	float:left;
	margin-right:10px;
	width:100px;
	height:88px;
	background-color:#000
}
.cornerBox table{
	padding:0;
	border-collapse:collapse;
	width:100%;
	height:100%;
}
.cornerBox table td.cornerBox_td{
	padding:0;
	vertical-align:middle;
	text-align:center;
}
.cornerBox table td.cornerBox_td .cornerImg.tl{position:absolute;top:0;left:0;}
.cornerBox table td.cornerBox_td .cornerImg.tr{position:absolute;top:0;right:0;}
.cornerBox table td.cornerBox_td .cornerImg.bl{position:absolute;bottom:0;left:0;}
.cornerBox table td.cornerBox_td .cornerImg.br{position:absolute;bottom:0;right:0;}
/*IE6中cornerBox的背景颜色会在图片底部多出一部分，因此通过图片下移的方法解决这个BUG，原因不明*/
* html .mainImg{margin-bottom:-4px;}

JS

<script type="text/javascript">
  window.onload = function(){
    var cornerBox = document.getElementsByClassName('cornerBox');
    for(i=0;i<cornerBox.length;i++){
      var cBox_td = cornerBox[i].getElementsByClassName('co0rnerBox_td');
      var corner_tl = document.createElement('span');
      var corner_tr = document.createElement('span');
      var corner_bl = document.createElement('span');
      var corner_br = document.createElement('span');
      corner_tl.innerHTML = '<img src="/images/css/corner_tl.gif" class="cornerImg tl" alt="cornerImg" />';
      corner_tr.innerHTML = '<img src="/images/css/corner_tr.gif" class="cornerImg tr" alt="cornerImg" />';
      corner_bl.innerHTML = '<img src="/images/css/corner_bl.gif" class="cornerImg bl" alt="cornerImg" />';
      corner_br.innerHTML = '<img src="/images/css/corner_br.gif" class="cornerImg br" alt="cornerImg" />';
      cBox_td[0].appendChild(corner_tl);
      cBox_td[0].appendChild(corner_tr);
      cBox_td[0].appendChild(corner_bl);
      cBox_td[0].appendChild(corner_br);
  }
}
</script>
