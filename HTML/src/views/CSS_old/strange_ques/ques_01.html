样式问题一

1.由于缺少”}”导致样式失效

.a_textM{ font-size:80%; color:#333333;}
.text1{font-size:70%; }
.text2{font-size:80%; letter-spacing:1px;}

如果a_textM缺少”}”,就会导致位于 a_textM后面的所有样式失效,此种情况只在FF中发生,IE6和IE7正常.

2.预防CSS注释BUG

中文注释BUG
在一些特殊情况下中文注释会导致部分代码失效,主要原因是服务器支持、文档编码类型以及程序类型等引起的.解决方法：
(一)将注释改成英文;
(二)更改文档编码类型;
HTML注释BUG
在内部样式中添加HTML注释,则后面相邻的样式也会失效;
