获取用户信息

uni.getUserProfile

uni.getUserProfile 只能放在按钮绑定函数中，不能放在其它函数中调用，即

					
                        //这样是不起作用的, 会报错: fail can only be invoked by user TAP gesture
                        uni.getUserProfile({
                            desc: "AAA",
                            lang: "zh_CN",
                            success:function(res){
                                console.log(res)
                            },
                            fail:function(res){
                                console.log(res)
                            }
                        })
