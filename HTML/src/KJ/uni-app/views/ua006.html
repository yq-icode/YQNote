导航栏背景图

为导航栏加背景图，首先要取消默认的原生导航栏，即设置 navigationStyle 为 custom, 然后再按下面的方法设置，即可实现导航栏背景图固定效果。

法一：page 背景图 + scroll-view

					
<template>
	<page>
		<uni-nav-bar fixed status-bar left-icon="back" @clickLeft="back" :title="pageTitle" color="#ffffff" background-color="transparent"></uni-nav-bar>
		<scroll-view scroll-y></scroll-view>
	</page>
</template>			
					
				
					
page {
	background-color: #EFF0F1;
	background-image: url('../static/images/bg2.jpg');
	background-repeat: no-repeat;
	background-position: center top;
	background-size: 100% 220rpx;
}
scroll-view {
	position: absolute;
	left: 0;
	right: 0;
	top: 220rpx;
	bottom: 0;
}
