中继器变量

pageIndex
当前页码
pageCount
总页数
visibleItemCount

中继器项目列表中可见项的数量。比如：项目列表共有15项，分页显示为每页6项。当项目列表在第1、2页时，可见项数量为6；当项目列表在第3页时，可见项数量为3。

当删除行时，visibleitemcount 值会立即变化；

itemCount
中继器项目列表的总数量，或者叫加载项数量。默认情况下项目列表的总数量会与中继器数据集中的数据行数量一致，但是，如果进行了筛选，项目列表的总数量则是筛选后的数量，这个数量不受分页影响。
dataCount
中继器数据集中数据行的总数量。
Item
获取数据集一行数据的集合，即数据行的对象。
TargetItem
目标数据行的对象。

元件属性

this
获取当前元件对象。当前元件是指当前添加交互动作的元件。
target
获取目标元件对象。目标元件是指当前交互动作控制的元件。
x, y
获取元件对象的X轴、Y轴坐标值。
width, height
获取元件对象的width、height值。
scrollX, scrollY
获取元件对象的水平、垂直滚动距离（当前仅限动态面板）
text
获取元件对象的文本文字。
name
获取元件对象的自定义名称。
top, left, right, bottom
获取元件对象的上边界、左边界、右边界、底边界坐标值。
opacity
获取元件对象的不透明比例。
rotation
获取元件对象的旋转角度。

鼠标属性

Cursor.x, Cursor.y
鼠标指针在页面中位置的X轴、Y轴坐标。
DragX、DragY
鼠标指针沿X轴、Y轴拖动元件时的瞬间(0.01秒)拖动距离。
TotalDragX、TotalDragY
鼠标指针拖动元件从开始到结束的X轴、Y轴移动距离。
DragTime
鼠标指针拖动元件从开始到结束的总时长。

数字

toFixed(decimalPoints)
将一个数字转为保留指定位数的小数，小数位数超出指定位数时进行四舍五入。decimalPoints为保留小数的位数。示例：如果n=1.232，[[n.toFixed(2)]]返回值1.23
toPrecision(length)
把数字示例化为指定的长度。length为示例化后的数字长度，小数点不计入长度。示例：如果n=1, [[n.toPrecision(6)]]返回值1.00000
toExponential(decimalPoints)
把对象的值转换为指数计数法。decimalPoints为保留小数的位数。示例：[[n.toExponential (参数)]]
