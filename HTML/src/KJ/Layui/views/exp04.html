为日历插件加图标

因为 layui.css 中 设置的 layui-input-block 为相对定位，且 input 宽度为 100%，所以为图标设置绝对定位，并为 input 设置 padding-right 空出图标的位置；

	                
<div class="layui-input-block">
	<input type="text" class="layui-input" id="YH-YXQ1" name="YXQ1" placeholder="年/月/日" style="padding-right: 30px;">
	<i class="iconfont icon-rili"></i>
</div>
		            
	            
	                
.layui-input-block .iconfont{
	position: absolute;
	right: 10px;
	top: 7px;
	color: #67767E;
}

如何将表单提交数据的格式由 名称值 对象模式 {name: value} 改成数组格式？

	                
<button type="button" id="btnSave" class="layui-btn layui-btn-sm btnC1" lay-submit="" lay-filter="dataSave">保存</button>
		            
	            
	                
form.on('submit(dataSave)', function(data){
	var arr = [];
	for(i in data.field){
		obj = {};
		obj[i] = data.field[i];
		arr.push(obj);
	}
	console.log(arr);
});
