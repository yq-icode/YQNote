checkbox的选中及部分选中效果

不通过点击设置checkbox的选中及部分选中效果

注意：先更新渲染，再设置部分选中，这样可以避免 部分选中效果 被 更新渲染覆盖。

	                   
$("input[type='checkbox']").each(function(){
	if($(this).attr('title') == '科目二'){
		var zob = $(this).parents('dt').next('dd').find("input[type='checkbox']");
		$(zob).each(function(){
			console.log($(this).index())
			if($(this).parents('.layui-col-xs3').index() < 5){
				$(this).attr('checked','');
			}
		})
		form.render('checkbox');//先更新渲染，再设置部分选中
		$(this).next('.layui-form-checkbox').addClass('checkedPart');
	}
})
