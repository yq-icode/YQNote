遇到的问题

tips超出页面

tips超出页面导致页面出现滚动条的问题，如下图

TIPS位置规律
  • 当TIPS方向为向上/下时，在靠近页面左边距时，TIPS与TD是左对齐的，在靠近页面右边距时，TIPS与TD是右对齐的；
  • 当TIPS方向为向左/右时，在靠近页面左边距时，TIPS紧挨TD右边框出现，在靠近页面右边距时，TIPS紧挨TD左边框出现；
  • 因此当TD隐藏部分超出页面边距时，就会导致页面滚动条出现
出现原因

上图设置的TIPS方向为向下，因此当鼠标移到TD上时，根据 "TIPS位置规律1"，由于TD只显示了一部分（隐藏部分已经超出了页面右边距），导致TIPS出现时，TIPS右边距也超出了页面右边距，从而致使页面现出滚动条。

解决方法
  • 将TIPS方向设置为向左或向右
  • 将TIPS宽度设置为：(document.width - td.width)/2
		                   
$("td").hover(
	function () {
		var tipWidth = ($(document).width() - $(this).outerWidth())/2 - 20;
    	idx = layer.tips("tips content", $(this), { tips: [2, "#333333"], time: 0, area: "auto", maxWidth: tipWidth });
	}, 
	function () {
    	layer.close(idx);
	}
)
