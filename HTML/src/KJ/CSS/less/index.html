默认参数 arguments

Less 可以使用默认参数，如果没有传参数，那么将使用默认参数。
@arguments 犹如 JS 中的 arguments， 指的是全部参数。
传的参数中 必须带着单位。

box-shadow

此写法只适合一组参数 
                    
    .box-shadow(@x:0; @y:0; @blur-radius:0; @spread-radius:0; @color:#ccc; @rest…){
        -webkit-box-shadow:@arguments; //指的是全部参数
        -moz-box-shadow:@arguments;
        box-shadow:@arguments;
    }
// 参数间用‘分号’间隔

注意：因为外阴影是默认投影方式，而且没有默认值；只有是内阴影时才会加上inset，所以关于投影方式这个变量不能随便起个变量名，只能用@rest...，而且必须得放在最后。内阴影参数值也必须得相应的放在最后，这样参数值和变量才是一一对应的。

                    
    .btn4{.box-shadow(0,3px);} //非零值必须带单位
    .btn4{.box-shadow(0,3px,0,0,#ccc);}
    .btn4{.box-shadow(0,3px,0,0,#ccc,inset);}
// 参数值用‘逗号’间隔；参数变量设置了默认值的可以省略不写；
此写法适用于一组及以上参数 
                    
    .box-shadow-multiple(@shadow){
        @shadow-double:@shadow;
        -webkit-box-shadow:@shadow;
        -moz-box-shadow:@shadow;
        box-shadow:@shadow;
    }            
                    
                
                    
    .btn4{
        ……
        @shadow:0 7px 0 0 #ccc, inset 0 8px 3px 0 rgba(0,0,0,.2);
        .box-shadow-multiple(@shadow);
    }

@rest

                    
//别忘记为@rest参数加'...'
.gradient-linear(@angle, @color_stop1, @color_stop2, @rest...){
    background: -webkit-linear-gradient(@angle, @color_stop1, @color_stop2, @rest);
    background: -moz-linear-gradient(@angle, @color_stop1, @color_stop2, @rest);
    background: -ms-linear-gradient(@angle, @color_stop1, @color_stop2, @rest);
    background: -o-linear-gradient(@angle, @color_stop1, @color_stop2, @rest);
}
//应用
#video_overlay {
    .gradient-linear(top,#000000,rgba(0,0,0,0.1) 80%,transparent);
}
