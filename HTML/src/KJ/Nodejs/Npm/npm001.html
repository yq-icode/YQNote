常用命令

安装与卸载

	            	
//指定配置文件
webpack --config webpack.dev.config.js

查看版本

	            	
/* ------------------------------ *
 * 查看安装路径 *
 * ------------------------------ */
where npm     //查看 npm 安装路径
where node    //查看 node 安装路径
where webpack //查看 webpack 安装路径

//查看某模块的详细信息 包括各版本号等
npm info packageName

//查看某模块版本号
webpack -v //查看webpack版本号

npm-init

npm-init
生成package.json文件
npm init -y / npm init --yes
直接略过所有问答，全部采用默认答案

安装&卸载

npm install sax@0.1.1
本地安装
npm install sax@0.1.1 -g
全局安装
npm install webpack --save-dev
为项目添加webpack依赖
npm uninstall sax@0.1.1
卸载
rimraf node_modules
删除 node_modules 文件夹
//先安装 npm 包 rimraf
npm install rimraf -g
//在 cmd 指令下，进入所需删除的 node_modules 文件夹的位置, 输入以下指令进行删除
rimraf node_modules

npm root（查看 node_modules 安装目录）

npm root
查看当前项目 node_modules 安装目录
npm root -g
查看全局 node_modules 安装目录

npm view

npm view packageName version
查看某模块远程最新版本
npm view packageName versions
查看某模块远程所有版本

npm list

npm list
列出当前项目安装的所有模块，以及它们依赖的模块
npm list -global
列出全局安装的模块
npm list packageName
列出单个模块
npm list -depth 0
列出依赖项深度（0：一级目录, 1: 一级目录和二级目录）
npm list -dev
列出 dependency 模块
npm list -prod
列出 production 模块
