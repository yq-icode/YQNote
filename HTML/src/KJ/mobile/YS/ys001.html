确认框

plus.nativeUI.confirm("info", callbackFunc(event), "title", [buttons])
info
弹出框显示的信息
callbackFunc(event)
点击弹窗框下方按钮后的触发事件
title
弹出框标题
buttons
弹出框下方放置的按钮组合
	                    
plus.nativeUI.confirm("显示的信息", function(event) {
	var tapIndex = event.index;
	switch (tapIndex) {
		case 0 : 
			console.log("你点击了'是'按钮");
			break;
		case 1 : 
			console.log("你点击了'否'按钮");
			break;
		case 2 : 
			console.log("你点击了'不确定'按钮");
			break;
	}
}, "标题", ["是", "否", "不确定"]);

警告框

plus.nativeUI.alert使用的是OS自带的原生警告框，在不同手机上也不一样。无法通过前端自定义样式。

plus.nativeUI.alert("info", alertFunc(event), "title", buttonName)
info
alertFunc(event)
title
定义标题，如果不需要标题，设置成 null
buttonName
自定义按钮文字

与confirm不同，alert框只用一个按钮，因此最后一个参数是一个String按钮名称，事件处理是通过alertFunc处理

	                    
plus.nativeUI.alert("提示的信息.", funtion(){}, "标题", "确定");

输入框

plus.nativeUI.prompt("info", promptFunc(event), "title", tips, [buttons])
info
输入对话框的提示
promptFunc
关闭对话框的回调
title
标题
tips
输入的地方显示的提示文字
buttons
对话框下显示的按钮数组

如何获取输入的值呢：通过promptFunc的回调函数中的event参数，event.index判断读取的哪一个按钮，event.value获取输入的值。

	                    
plus.nativeUI.prompt("输入一个数字", function(event){
	if(event.index){ //确定按钮
		console.log("你输入了：" + event.value);
	} else { //取消按钮
		console.log("你取消了该次输入");
	}
}, "标题", "输入提示", ["取消", "确定"]);

弹出自动消失的浮动框

plus.nativeUI.toast("info", options)
info
弹出的内容信息
options
对于弹出框的一些说明选项信息，可以配置弹出框显示的图标、位置以及持续时间。
  • icon: 显示的图标
  • duration: 消息框持续时间(long/short)
  • align: 水平方向位置(left/center/right)
  • verticalAlign: 垂直方向位置(top/center/bottom)
	                    
plus.nativeUI.toast("弹出的消息内容", {
	icon : "icon URL", //eg. "/img/add.png"
	duration : "long", //持续3.5s，short---2s
	align : "center",
	verticalAlign : "bottom"
})
